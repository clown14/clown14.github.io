### DFS(G)

``` bash
for each vertex u∈G.V 图G=(V,E)
{
  u.color=WHITE
  u.π=NULL	（u前驱
}
time=0
for each vertex u∈G.V
{
  if(u.color==WHITE)
  {
    DFS-VISIT(G,u)
  }
} 
```

### DFS-VISIT(G,u)
``` bash
time++
u.d=time (发现时间
u.color=GRAY
for each v∈G:Adj[u]  (邻接链表
{
  if(v.color==WHITE)
  {
    v.π=u
    DFS-VISIT(G,v)
  }
} 
u.color=BLCAK
time++
u.f=time (结束时间
```
